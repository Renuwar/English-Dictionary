Computer Dictionary

Algorithm

An algorithm is a set of instructions designed to perform a specific task. In computer science, algorithms are often used for data processing and mathematical calculations.

Binary

Binary refers to the base-2 number system, which is used in computers to represent numeric values using only two symbols: 0 and 1.

Compiler

A compiler is a software tool that translates source code written in a high-level programming language into machine code that can be executed by a computer.

Database

A database is an organized collection of data, typically stored and accessed electronically from a computer system. Databases can be relational, hierarchical, or distributed, among other types.

Encryption

Encryption is the process of encoding data in such a way that only authorized parties can access it. It is commonly used to protect sensitive information during transmission and storage.

Firewall

A firewall is a network security device or software that monitors and controls incoming and outgoing network traffic based on predetermined security rules. It acts as a barrier between a trusted internal network and untrusted external networks, such as the internet.

GUI (Graphical User Interface)

A GUI is a type of user interface that allows users to interact with electronic devices using graphical icons and visual indicators, as opposed to text-based interfaces, typed command labels, or text navigation.

HTML (Hypertext Markup Language)

HTML is the standard markup language for creating web pages and web applications. It defines the structure and layout of a web document by using various tags and attributes to describe the content.

Input/Output (I/O)

Input/Output refers to the communication between a computer and external devices, such as keyboards, mice, monitors, printers, and storage devices. Input devices send data to the computer, while output devices receive data from the computer.

JavaScript

JavaScript is a programming language commonly used to create interactive effects within web browsers. It is widely used for client-side web development, allowing developers to add dynamic content and interactivity to websites.

Kilobyte (KB)

A kilobyte is a unit of digital information storage that equals 1,024 bytes. It is commonly used to measure the size of small files and data storage capacities.

Laptop

A laptop, also known as a notebook computer, is a portable personal computer designed for mobile use. It typically includes a display screen, keyboard, touchpad or pointing stick, and battery in a single compact unit.

Machine Learning

Machine learning is a subset of artificial intelligence that focuses on the development of algorithms and statistical models that enable computers to learn and make predictions based on data rather than being explicitly programmed.

Network

A network is a group of interconnected computers and other devices that can communicate with each other to share resources, such as files, printers, and internet connections. Networks can be classified based on their geographical scope, such as local area networks (LANs), wide area networks (WANs), and the internet.

Operating System (OS)

An operating system is system software that manages computer hardware, software resources, and provides common services for computer programs. It acts as an intermediary between users and the computer hardware, allowing users to interact with the hardware and run applications.

Pixel

A pixel, short for picture element, is the smallest unit of display on a computer screen or digital image. It is a single point in a raster image, which consists of a grid of pixels arranged in rows and columns. Pixels are used to represent and display images, graphics, and text on computer screens and digital devices.

Query

A query is a request for information from a database or search engine, typically expressed in a specific query language, such as SQL (Structured Query Language) for databases or a search query for search engines. Queries are used to retrieve, manipulate, and analyze data stored in a database or indexed by a search engine.

RAM (Random Access Memory)

RAM is a type of computer memory that is used to store data and machine code currently being used or processed by the CPU (central processing unit). It is volatile memory, meaning that it requires power to maintain stored data, and data is lost when the power is turned off or the computer is restarted.

Server

A server is a computer or software system that provides resources, services, or data to other computers, known as clients, over a network. Servers can serve various purposes, such as hosting websites, managing network resources, storing and sharing files, and processing client requests.

Terabyte (TB)

A terabyte is a unit of digital information storage that equals 1,024 gigabytes or approximately one trillion bytes. It is commonly used to measure the storage capacity of hard disk drives, solid-state drives, and other data storage devices.

Unix

Unix is a family of multitasking, multiuser computer operating systems that derive from the original AT&T Unix, developed in the 1970s. Unix systems are widely used in servers, workstations, and mobile devices due to their stability, security, and flexibility.

Virtual Reality (VR)

Virtual reality is a simulated experience that can be similar to or completely different from the real world. It uses computer technology to create a three-dimensional environment that users can interact with and explore, often through immersive head-mounted displays and sensory feedback devices.

Wi-Fi

Wi-Fi, short for Wireless Fidelity, is a wireless networking technology that allows devices to connect to the internet and communicate with each other wirelessly within a local area network (LAN). Wi-Fi networks use radio waves to transmit data between devices, providing convenient and flexible connectivity.

XML (Extensible Markup Language)

XML is a markup language that defines a set of rules for encoding documents in a format that is both human-readable and machine-readable. It is widely used for representing and exchanging structured data over the internet, such as configuration files, data interchange between applications, and web services communication.

Algorithm

An algorithm is a set of instructions designed to perform a specific task. In computer science, algorithms are often used for data processing and mathematical calculations.

API (Application Programming Interface)

An API is a set of rules and protocols that allows different software applications to communicate with each other. It defines the methods and data formats that applications can use to request and exchange information, enabling seamless integration and interoperability between systems.

Bit

A bit is the smallest unit of data in a computer and represents a binary digit, which can have a value of either 0 or 1. Bits are the building blocks of digital information and are used to represent and process data in binary form within computer systems.

Cloud Computing

Cloud computing is a technology that allows users to access and use computing resources, such as servers, storage, databases, and software applications, over the internet. It eliminates the need for local hardware and software infrastructure, enabling on-demand access to scalable and flexible computing resources.

Debugging

Debugging is the process of identifying and fixing errors, bugs, and other issues in software programs or computer hardware. It involves analyzing the program's behavior, isolating the root cause of the problem, and making corrections to ensure proper functioning.

Executable File

An executable file is a type of computer file that contains machine code or instructions that can be executed by a computer's operating system. It typically has a file extension such as .exe on Windows or .app on macOS and is used to launch and run software applications.

Firmware

Firmware is a type of software that is permanently stored in a hardware device's read-only memory (ROM) or flash memory. It provides low-level control and operational instructions for the device's hardware components and is responsible for booting the device and controlling its basic functions.

Graphics Processing Unit (GPU)

A graphics processing unit (GPU) is a specialized electronic circuit designed to accelerate the rendering of images and videos in a computer's display system. GPUs are commonly used in gaming, multimedia, and other graphics-intensive applications to improve performance and visual quality.

Hyperlink

A hyperlink is a selectable text, image, or other element in a web page that, when clicked, navigates to another location within the same document or to a different document or resource on the internet. Hyperlinks are fundamental to the concept of hypertext and enable non-linear navigation between web pages and resources.

JavaScript Object Notation (JSON)

JSON is a lightweight data interchange format that is easy for humans to read and write and easy for machines to parse and generate. It is commonly used for transmitting data between a web server and a web client as an alternative to XML (Extensible Markup Language).

Malware

Malware is a type of malicious software designed to disrupt, damage, or gain unauthorized access to computer systems and networks. Examples of malware include viruses, worms, trojans, ransomware, spyware, and adware.

Open Source

Open source refers to software that is released with its source code under a license that allows users to study, modify, and distribute the software for any purpose. Open source software promotes collaboration, transparency, and community-driven development.

Peripheral

A peripheral is an external device connected to a computer that extends its capabilities or provides additional functionality. Common peripherals include keyboards, mice, printers, monitors, speakers, and external storage devices.

Queue

A queue is a data structure that follows the First In, First Out (FIFO) principle, where the first element added to the queue is the first one to be removed. It is often used in computer science for managing tasks, scheduling processes, and implementing breadth-first search algorithms.

Root Directory

The root directory is the top-level directory in a hierarchical file system, from which all other directories and files descend. In Unix-based systems, the root directory is denoted by a forward slash (/), while in Windows, it is typically represented by the drive letter followed by a backslash (e.g., C:\).

Stack

A stack is a data structure that follows the Last In, First Out (LIFO) principle, where the last element added to the stack is the first one to be removed. It is commonly used in computer science for managing function calls, storing temporary data, and implementing depth-first search algorithms.

Transaction

A transaction is a unit of work or a series of operations performed on a database that must be executed as a single, indivisible unit. It follows the ACID properties (Atomicity, Consistency, Isolation, Durability) to ensure data integrity and reliability in database management systems.
